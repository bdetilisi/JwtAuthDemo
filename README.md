# JwtAuthDemo
JwtAuthDemo is a sample ASP.NET Core Web API that demonstrates how to implement JWT-based authentication using Entity Framework Core for user management and data persistence.

## Features
This project demonstrates how to:
- Configure JWT Bearer authentication in ASP.NET Core
- Generate and validate JWT access tokens
- Implement refresh token support for secure session renewal
- Use Entity Framework Core for user storage and data access
- Secure API endpoints with role-based authorization policies
- This sample provides a practical foundation for building secure, token-based authentication in ASP.NET Core applications.
